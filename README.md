# MyFavoritePlaces

The iOS application is a list of favorite places.
They can be added, deleted, sorted and rated places, set the location on the map and build the nearest routes.

Used in the project:
- UITableView
- UISearchController
- UIImagePicker
- MapKit
- Realm
- Cosmos
- MVC
